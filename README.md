# exotic-api

- $ npm install
- create database
- write bdd credentials (in localConfif object) from /conf.js (isProd = false)
- insert last datas from /dump/xx-xx-xx.sql
- $ node api.js
- You can use these fakes token for API testing : token1, token2, token2... token10
